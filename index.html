O que é o projeto de educação financeira?

Esse projeto é para mostrar como as pessoas gastam dinheiro de forma desnecessária ou como gastam muito dinheiro só para esbanjar sua riqueza para outras pessoas, tentando se mostrar “superior”.

Quais as disciplinas envolvidas no projeto e o que nós fizemos nelas?

  • Sonivaldo - Matemática: Cálculos para gastos de dinheiro;
  • Mariza - Português: História em quadrinhos;
  • Eric - Projeto Design Web: Ordem das informações nas páginas;
  • Eric - Front-End: Web pronta para postar.
